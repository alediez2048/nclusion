# OPS-003: Add settlement audit view for a match

**Phase:** 3 — Demo Hardening
**Status:** ⏳ Pending

## Description

Show all bets for a match, settlement result, payout summary. Display settlement tx signature. Useful for evaluator demo and support investigation.

## Acceptance Criteria

- [ ] Each bet shows outcome (won/lost/cancelled)
- [ ] Payout summary totals displayed
- [ ] Settlement tx signature shown
- [ ] Accessible from match search
