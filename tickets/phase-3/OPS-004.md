# OPS-004: Add manual settlement hold flag for disputed results

**Phase:** 3 — Demo Hardening
**Status:** ⏳ Pending
**Priority:** High

## Description

Operator can flag a match to prevent automatic settlement. Settlement service checks for hold flag before processing. Operator can clear the flag to resume settlement.

## Acceptance Criteria

- [ ] Hold flag settable by operator per match
- [ ] Settlement service respects hold flag
- [ ] Held matches show "Settlement paused" status
- [ ] Operator can clear flag to resume
- [ ] Users see "Settlement delayed — under review" message

- **Blocked by:** OPS-001, SET-002
