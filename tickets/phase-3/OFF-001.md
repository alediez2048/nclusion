# OFF-001: Cache match lists and top match details locally

**Phase:** 3 — Demo Hardening
**Status:** ⏳ Pending
**Priority:** High

## Description

Store match list and top match details in op-sqlite on each successful fetch. Serve from cache when offline with freshness indicator. TTL-based expiry for stale data.

## Acceptance Criteria

- [ ] Matches cached in op-sqlite after fetch
- [ ] Cached matches viewable when offline
- [ ] Freshness label shown on cached data
- [ ] Stale data expires based on TTL
- [ ] Cache updates on successful reconnect
