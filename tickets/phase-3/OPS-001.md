# OPS-001: Create lightweight operator dashboard for bet lookup

**Phase:** 3 — Demo Hardening
**Status:** ⏳ Pending
**Priority:** High

## Description

Static SPA deployed to Cloudflare Pages. Search bets by user ID, bet ID, or match ID. Show bet record with full lifecycle details. Authenticated as operator role.

## Acceptance Criteria

- [ ] Dashboard loads and authenticates operator
- [ ] Search by user ID, bet ID, or match ID
- [ ] Bet detail shows full lifecycle with timestamps
- [ ] Deployed to Cloudflare Pages
- [ ] Calls backend API with operator auth
