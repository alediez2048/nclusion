# UX-002: Add status chips and progress messaging for bet lifecycle

**Phase:** 3 — Demo Hardening
**Status:** ⏳ Pending
**Priority:** Medium

## Description

Color-coded status chips: Draft (gray), Queued (yellow), Processing (blue), Confirmed (green), Pending settlement (orange), Won (green), Lost (red), Cancelled (gray), Failed (red). Brief progress message under each.

## Acceptance Criteria

- [ ] All 9 statuses have distinct chip color
- [ ] Progress message shown below each chip
- [ ] Messages localized in Creole and French
- [ ] Chips render correctly on small screens
- [ ] Contrast ratio accessible
