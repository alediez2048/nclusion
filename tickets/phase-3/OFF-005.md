**Phase:** 3 — Demo Hardening
**Status:** ⏳ Pending

## Description

- [ ] Insufficient balance: "Not enough HTGN available."
- [ ] Chain delay: "Your bet is being processed. This may take a moment."
- [ ] Each message has clear next-step guidance
- [ ] All messages localized in Creole and French
