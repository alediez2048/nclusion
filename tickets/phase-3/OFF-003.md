# OFF-003: Allow draft bet intent creation under weak connectivity

**Phase:** 3 — Demo Hardening
**Status:** ⏳ Pending
**Priority:** High

## Description

User can build a bet slip and save as Draft even when offline. Draft stored in op-sqlite with short expiration window. Clearly labeled "Not submitted" in UI.

## Acceptance Criteria

- [ ] Bet slip saveable as draft when offline
- [ ] Draft stored in op-sqlite with expiration
- [ ] "Not submitted" label clearly visible
- [ ] Expired drafts auto-removed
- [ ] Draft accessible from home screen
