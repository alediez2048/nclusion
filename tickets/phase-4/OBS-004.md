# OBS-004: Add alerting for stuck bets and failed settlement batches

**Phase:** 4 — MVP Operational Hardening
**Status:** ⏳ Pending
**Priority:** High

## Description

Alert when a bet stays in Processing > N minutes. Alert when settlement batch fails. Notification via email or Slack webhook.

## Acceptance Criteria

- [ ] Stuck bet alert fires after configurable threshold
- [ ] Failed settlement alert fires on batch failure
- [ ] Notifications delivered (email or Slack)
- [ ] Alert includes bet/settlement details for investigation
- [ ] Thresholds configurable
