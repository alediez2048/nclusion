# RPC-001: Add multi-provider RPC failover

**Phase:** 4 — MVP Operational Hardening
**Status:** ⏳ Pending
**Priority:** High

## Description

Configure multiple Solana RPC providers in relay service. Automatic failover when primary provider fails or times out. Transparent to callers.

## Acceptance Criteria

- [ ] Multiple RPC providers configured
- [ ] Failover triggers automatically on failure/timeout
- [ ] Transactions still submit during provider outage
- [ ] No user-visible disruption during failover
- [ ] Failover events logged
