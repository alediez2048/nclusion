# SET-008: Add market cancellation path

**Phase:** 4 — MVP Operational Hardening
**Status:** ⏳ Pending
**Priority:** Medium

## Description

Cancel a market when external factors prevent fair settlement (match abandoned, data corruption). Return all escrowed funds to bettors. Bets marked Cancelled.

## Acceptance Criteria

- [ ] Market cancellable by authorized operator
- [ ] All escrowed funds returned to bettors
- [ ] All bets marked Cancelled
- [ ] Balance service updates user balances
- [ ] Cancellation logged with reason
