# SET-006: Add secondary results provider cross-check

**Phase:** 4 — MVP Operational Hardening
**Status:** ⏳ Pending
**Priority:** High

## Description

Settlement service checks primary result against a secondary provider. Proceed only when both agree. Hold for manual review if they disagree.

## Acceptance Criteria

- [ ] Secondary provider integrated
- [ ] Results cross-checked before settlement
- [ ] Matching results proceed to settlement
- [ ] Disagreements hold the market
- [ ] Held markets queued for manual review
