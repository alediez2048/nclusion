# SET-007: Add disputed result workflow

**Phase:** 4 — MVP Operational Hardening
**Status:** ⏳ Pending
**Priority:** High

## Description

When providers disagree, market enters "disputed" state. Users see "Settlement delayed — under review." Operator can resolve dispute and trigger settlement.

## Acceptance Criteria

- [ ] Disputed markets enter held state
- [ ] Users see delay messaging
- [ ] Operator can view dispute details
- [ ] Operator can resolve and trigger settlement
- [ ] Resolution logged for audit
