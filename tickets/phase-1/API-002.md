# API-002: Create API gateway with auth middleware

**Phase:** 1 — Foundations
**Status:** ⏳ Pending
**Priority:** High

## Description

Create Hono app in apps/gateway. Add auth middleware (JWT or session-based), rate limiting, idempotency key handling for mutation requests, and request logging.

## Acceptance Criteria

- [ ] Gateway starts and serves health check
- [ ] Auth middleware rejects unauthenticated requests
- [ ] Rate limiting returns 429 on excess requests
- [ ] Idempotency key deduplication works on POST endpoints
- [ ] Request logging emits structured JSON
