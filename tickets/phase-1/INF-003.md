# INF-003: Set up CI for lint, test, and build checks

**Phase:** 1 — Foundations
**Status:** ⏳ Pending
**Priority:** Medium

## Description

GitHub Actions workflows: lint (Biome + clippy), type check, test (Vitest + cargo test), build. Turborepo --filter for incremental checks on PRs.

## Acceptance Criteria

- [ ] CI runs on every PR
- [ ] Lint job runs Biome (TS) and clippy (Rust)
- [ ] Type check job runs tsc
- [ ] Test job runs Vitest and cargo test
- [ ] Build job runs turbo build
- [ ] Incremental filtering via --filter=[HEAD^1]
- [ ] All checks pass on clean repo

## Dependencies

- **Blocks:** None (enables safe iteration)
- **Blocked by:** API-001, SOL-001
