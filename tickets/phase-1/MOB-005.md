# MOB-005: Add network-state detection and connectivity banner

**Phase:** 1 — Foundations
**Status:** ⏳ Pending
**Priority:** Medium

## Description

Detect network state (offline, limited connection, synced) and display a persistent banner when not fully connected. Adjust app behavior: disable write operations when offline, show cached data with freshness indicators.

## Acceptance Criteria

- [ ] Network state detected accurately (offline/limited/synced)
- [ ] Persistent banner shown for offline and limited states
- [ ] Write operations disabled when offline
- [ ] Cached data served with freshness indicator
- [ ] State transitions smooth (no flicker)
