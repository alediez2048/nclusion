# API-003: Create user service and managed wallet mapping

**Phase:** 1 — Foundations
**Status:** ⏳ Pending
**Priority:** High

## Description

User registration/login with phone-based or lightweight identity. Map users to managed wallet records. Store user preferences (language). No seed phrase exposure.

## Acceptance Criteria

- [ ] User can register and login
- [ ] Managed wallet mapping created on registration
- [ ] User profile includes language preference
- [ ] Wallet public key stored and retrievable
- [ ] No seed phrase or private key exposed to client

## Dependencies

- **Blocks:** WAL-001
- **Blocked by:** API-001, API-002
