# BET-001: Build match detail UI and selection flow

**Phase:** 2 — Core Betting Loop
**Status:** ⏳ Pending
**Priority:** High

## Description

Mobile screen showing match info (teams, time, status) and outcome options (Home / Draw / Away). Tap an outcome to open the bet slip. Display current odds for each option.

## Acceptance Criteria

- [ ] Match detail screen renders team info and match time
- [ ] Three outcome buttons with current odds
- [ ] Tapping outcome navigates to bet slip
- [ ] Selected outcome passed to bet slip
- [ ] Loading and error states handled
