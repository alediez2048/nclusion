# BAL-002: Add reconciliation job for chain and ledger views

**Phase:** 2 — Core Betting Loop
**Status:** ⏳ Pending
**Priority:** Medium

## Description

Periodic job comparing shadow ledger balances against on-chain SPL token balances. Flag mismatches for investigation. Never auto-correct — log and alert only.

## Acceptance Criteria

- [ ] Reconciliation job runs on schedule
- [ ] Compares shadow ledger vs on-chain balance per user
- [ ] Mismatches detected and logged with details
- [ ] No automatic corrections
- [ ] Alert mechanism for mismatches
