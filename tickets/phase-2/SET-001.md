# SET-001: Integrate result provider

**Phase:** 2 — Core Betting Loop
**Status:** ⏳ Pending
**Priority:** High

## Description

Connect to sports data provider for final match results. Poll for completed matches or handle webhook if available. Parse result data into internal format.

## Acceptance Criteria

- [ ] Provider connected for match results
- [ ] Completed match results fetched/received
- [ ] Results parsed into internal format
- [ ] Polling interval or webhook configured
- [ ] Error handling for provider failures
