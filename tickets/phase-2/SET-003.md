# SET-003: Implement settle_market instruction

**Phase:** 2 — Core Betting Loop
**Status:** ⏳ Pending
**Priority:** High

## Description

Anchor instruction: takes market_id and result. Settles all bets in market — marks as won/lost/cancelled. Releases escrow: pays winners from escrow, returns remainder to platform.

## Acceptance Criteria

- [ ] Instruction settles all bets in a market
- [ ] Winners receive correct payout from escrow
- [ ] Losers marked as lost, no payout
- [ ] Escrow account emptied after settlement
- [ ] Settlement authority enforced (only authorized signer)
- [ ] bankrun tests cover all outcomes
