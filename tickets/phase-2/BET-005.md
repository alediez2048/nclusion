# BET-005: Validate stake, odds window, and balance before submission

**Phase:** 2 — Core Betting Loop
**Status:** ⏳ Pending
**Priority:** High

## Description

Before processing a bet intent: check market is still open, odds within tolerance of displayed odds, sufficient available balance, stake within limits. Reject with specific error codes.

## Acceptance Criteria

- [ ] Market availability checked
- [ ] Odds tolerance validated (reject if moved too far)
- [ ] Balance sufficiency checked
- [ ] Stake limits enforced
- [ ] Specific error codes for each failure type
- [ ] All validations tested
