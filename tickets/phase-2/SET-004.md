# SET-004: Update balances after settlement

**Phase:** 2 — Core Betting Loop
**Status:** ⏳ Pending
**Priority:** High

## Description

After on-chain settlement confirmed, update shadow ledger. Winners: move from pending_settlement to available with winnings added. Losers: remove pending_settlement amount.

## Acceptance Criteria

- [ ] Winner balance: pending_settlement → available + winnings
- [ ] Loser balance: pending_settlement removed
- [ ] Balance updates atomic
- [ ] Reconciliation with on-chain state after settlement
- [ ] Tested for multiple bets in one settlement
