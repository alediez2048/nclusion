# SOL-007: Implement escrow logic for HTGN stake

**Phase:** 2 — Core Betting Loop
**Status:** ⏳ Pending
**Priority:** High

## Description

Transfer HTGN from user wallet to market escrow PDA token account on bet placement. Escrow holds funds until settlement. Uses SPL token transfer CPI.

## Acceptance Criteria

- [ ] HTGN transferred from user to escrow on place_bet
- [ ] User token balance decreases by stake
- [ ] Escrow token balance increases by stake
- [ ] SPL token program CPI invoked correctly
- [ ] bankrun tests verify token transfers
