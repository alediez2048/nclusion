# BET-007: Generate deterministic bet ID

**Phase:** 2 — Core Betting Loop
**Status:** ⏳ Pending
**Priority:** High

## Description

Derive bet ID deterministically from intent_id + market_id + user_id. Same inputs always produce same bet ID. Used for on-chain PDA derivation and duplicate prevention.

## Acceptance Criteria

- [ ] Deterministic generation: same inputs → same ID
- [ ] No collisions across different input combinations
- [ ] Compatible with Solana PDA seed requirements
- [ ] Used by both backend and Solana program

## Dependencies

- **Blocks:** SOL-005, REL-001
- **Blocked by:** BET-004
