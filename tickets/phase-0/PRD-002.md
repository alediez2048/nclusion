# PRD-002: Finalize supported betting markets for demo: win/draw/loss only

**Phase:** 0 — Discovery and Scope Lock
**Status:** ⏳ Pending
**Priority:** High

## Description

Lock the bet types supported in the MVP to single bets on match outcome (Home win / Draw / Away win). No parlays, props, over/under, or live betting. This constrains the Solana program design, bet slip UI, and settlement logic.

## Acceptance Criteria

- [ ] Market types locked to 1X2 (win/draw/loss)
- [ ] Bet slip logic scoped to single bets only
- [ ] Settlement logic scoped to single-outcome resolution

- **Blocked by:** PRD-001
