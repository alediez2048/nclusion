# PRD-003: Define user-visible bet statuses and balance labels

**Phase:** 0 — Discovery and Scope Lock
**Status:** ⏳ Pending
**Priority:** High

## Description

Finalize the bet state machine for both user-facing and internal states, and lock the balance label terminology.

User-facing states: Draft, Queued, Processing, Confirmed, Pending settlement, Won, Lost, Cancelled, Failed.

Internal states: intent_created, validation_passed, funds_reserved, relay_submitted, signature_received, chain_confirmed, awaiting_result, settlement_submitted, settlement_confirmed, reverted_or_released.

Balance labels: Available, In active bets, Pending settlement.

## Acceptance Criteria

- [ ] User-facing state machine documented with transitions
- [ ] Internal state machine documented with transitions
- [ ] Balance labels finalized
- [ ] Haitian Creole translations drafted for all user-facing terms
